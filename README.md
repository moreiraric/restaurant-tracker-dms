# Restaurant Tracker DMS

Restaurant Tracker DMS is a Java command-line program that allows users to manage restaurant records. Users can add, view, update, delete, load, and summarize restaurant data through a simple text-based menu.

## Features

- Add new restaurant records
- View all saved restaurants
- Update existing restaurant records
- Delete restaurant records
- Load restaurant data from a text file
- Generate a summary report
- Validate user input for menu choices, ratings, price levels, yes/no answers, and dates

## Restaurant Data Fields

Each restaurant record includes:

- Restaurant name
- Cuisine type
- Location
- Price level
- User rating
- Visited status
- Date visited
- Notes
